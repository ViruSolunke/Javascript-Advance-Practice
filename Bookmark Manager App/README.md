## 📖 About The Project

The **Bookmark Manager App** is a simple and interactive web application that allows users to save, organize, view, and delete bookmarks based on categories.

The application uses the browser's **Local Storage API**, ensuring that all bookmarks remain available even after refreshing or reopening the browser.

This project was built to practice:

* DOM Manipulation
* Event Handling
* Local Storage
* Array Methods
* Dynamic UI Rendering
* JavaScript Application Development

---

## ✨ Features

✅ Add bookmarks with name and URL.

✅ Organize bookmarks by categories.

✅ View bookmarks category-wise.

✅ Delete bookmarks instantly.

✅ Data persistence using Local Storage.

✅ Responsive and clean user interface.

✅ Dynamic rendering without page refresh.
