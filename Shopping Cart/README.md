# ⚙️ How It Works

### Add to Cart

Click the **Add to Cart** button to add any dessert.

---

### Quantity Counter

If the same item is added multiple times, the quantity updates automatically.

Example:

```
2x Chocolate Cupcake
```

---

### Automatic Billing

The application calculates

- Subtotal
- Taxes (8.25%)
- Grand Total

without refreshing the page.

---

### Clear Cart

Click **Clear Cart**

✔ Confirmation popup

✔ Removes all items

✔ Resets totals

✔ Clears product list
